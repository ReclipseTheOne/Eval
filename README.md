<div align="center">
 <h1>Configuration</h1>
</div>


The direct config file is located at config/eval.json and can be hot reloaded via <code>/eval reload</code>
Show the current eval config via <code>/eval status</code>

The format pattern can be changed via <code>/eval start 'pattern'</code> and <code>/eval end 'pattern'</code>

And inlining can be turned on and off via <code>/eval inline on/off</code>
